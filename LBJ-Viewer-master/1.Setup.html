Installation

To install the system, you need to do the following steps:

1. Download the source files

  • Download the installation package and extract the files to be base website directory a. The path should be ~/catalina/webapps/OnlineImageProcess

2. Tomcat

  • Inside the terminal, execute the commands in order:

  • a. starttomcat

  • b. javac WEB-INF/classes/*.java

  • c. cd ~/catalina/webapps/OnlineImageProcess/SQLstatement

  • d. sqlplus ,

  • e. @setupscript.sql

  • f. starttomcat

3. Connect

  • In the web browser, enter: http://u***.cs.ualberta.ca:/OnlineImageProcess/welcome.html where "u***" is the computer number that you are using.
