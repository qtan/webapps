3.1 Create Groups

One cannot login without providing a valid username and password.

After a successful login, you can create any number of groups by clicking the create groups button.

  • You can create your group name yourself by simply typing your desired group name into the input box labelled Group Name
  • You can then input a notice for you group by typing input into the text box labelled Notice
  • A friend can then be added to the group if the person is already a registered user, and is not in the group currently.
  • This is achieved by checking the checkbox next to that friend's username
  • If you made a mistake anywhere or wish to clear your inputs simply press the reset button given.
  • Otherwise, feel free to click the Create button to create your desired group.
  • *A user can add more friends to the group after the group is created or a friend is deleted from the group.

3.2 View groups

  • View groups is a section accessed after groups are created from the profile page.

  • From this page you can specify the permission of the each image in the group by selecting options in the drop down box which will contain public/private and group only

  • Public means all the users can view the image.

  • Private means only owner and administrator can view and edit the image.

  • Group means only the users in the group can view the image.

  • * For updating the information, only owner and administrator has the authority. Keep in mind that administrator can view and edit anything in the system.
